# tesla-mongo-connect

An addon to [tesla-microservice](https://github.com/otto-de/tesla-microservice)
that allows to read an write to mongodb.

## Usage

Add this to your project's dependencies:

`[de.otto/tesla-mongo-connect "0.1.0"]`
